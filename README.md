# Smart Traffic Light Simulator 🚦

This is a simple Python project that simulates a traffic signal system.  
The program takes the number of cars waiting and the time of day as input and decides the green light duration.

If there are many cars, the green light stays longer.  
If there are very few cars, the green light stays shorter.  
At night, the signal changes to blinking yellow mode.

This project demonstrates basic Python concepts such as user input, comparison operators, and if-elif conditions.
